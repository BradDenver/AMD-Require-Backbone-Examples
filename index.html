Bootstrap Require Backbone Examples

This site is designed to be a guide to building web apps using a stack including Twitter Bootstrap, RequireJS and Backbone.js. While there are many great articles around already for each of these tools in isolation this site will focus on developing for this combined stack. By sticking to this set stack I hope to provide value through detailed examples, and to remove the "how do I relate that to my dev stack?" questions I often have when reading more general guides.

So whats in my choosen dev stack?

Twitter Bootstrap
will provide us with a css base
jQuery
will be our library of choice for working with the DOM
RequireJS
will handle our javascript loading and dependency management
Backbone.js
is the framework we will use to build our application.
Underscore.js
is a requirement for Backbone.js and contains some great javascript utilites
Backbone localStorage
